# Session Handover — 2026-06-30 (exp29 multi-epoch deposition emulator)

## Goal
Turn the exp25/26 deposition kernel into a **phenomenological forward-model emulator**:
halo MAH (+ halo-only secondaries) → stellar-mass **CoG / profile at 5 epochs**
(z=0.4/0.7/1.0/1.5/2.0). Keep it flexible, fast, mathematically reasonable; physical
plausibility is *not* required. Use **CoG** as the target (it fits best).

## In Progress (Not Finished) — THE NEXT TASK
**Independent single-epoch fits to EVERY snapshot, especially z=2** — to test whether the
centred-Gaussian deposit has a *fundamental* limit for high-z massive galaxies.
- Per galaxy, per epoch z_k: fit deposits up to t(z_k) to *that epoch's CoG alone*,
  normalized to M*(z_k). Reuse `cog_extrapolate.py` / `single_vs_multi.py` (change the target epoch).
- **Decision it settles:** if z=2 single-epoch fits are as good as z=0.4 (~0.004 dex / 2%)
  → the deposit *shape* is fine; the only problem is multi-epoch *consistency* → **build the
  puff-up model**. If z=2 fits **fail** for massive galaxies → the deposit *shape itself* is
  the limit → rethink the high-z primitive.
- Evaluate in **linear M*** + relative residual, report **max / 90th-percentile** relative
  error (not just mean log-RMS), **exclude inner ≤3 kpc** (softening). Stratify by mass.
- Full spec: bottom of `experiments/exp29_outer_deposit/PUFF_MODEL_PLAN.md`.

## Problems / Blockers
- **Structural limit** above — the open scientific question this exp must resolve.
- **Metric**: mean log-RMS understates per-galaxy error; always pair with radius×epoch
  relative residuals + percentiles, exclude inner ≤3 kpc.
- **Aperture leak**: Gaussian deposits put ~8% mass beyond 148 kpc → model CoG ~0.035 dex
  low at the outer radius (pin normalization to the 148-kpc aperture to fix).

## Key Decisions
- **Phenomenological emulator, not SFH** — `f` is a deposit fraction, no physical recipe.
- **Forward inputs must be halo-only** — M*, R50 are illegal (predicted, not given);
  `c_200c` substitutes for R50 almost perfectly. M* comes from a separable SHMR.
- **CoG, not Σ** as the primary target (fits best; user approved).
- **Free Sérsic rejected** (bad math); **puff-up parked** (one extra knob, deferred until
  the high-z single-epoch diagnostic justifies it).
- Compute lever: CoG linear in masses → **NNLS** convex inner solve; cache fits to npz.
