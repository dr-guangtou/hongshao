- **exp26 — differential stellar-density profiles** (commit `1e36975`, merge
  `9574008`): built/stored `ΔΣ(R)` between adjacent z epochs (z=0.4/0.7/1.0/1.5/2.0)
  for the full profile-valid sample (**n=3380**). **The added mass is NOT a centred

  core-drop). → the kernel needs a multiplicative / outer-weighted primitive, not a
  centred Gaussian. Files: `differential_profiles.py` (+ `declining_mah.py`,
  `declining_growth.py`, `declining_properties.py`).
- **Declining-MAH investigation:** the 695 `mah_declined` are all **centrals** (0
  satellites); 4 archetypes (mostly benign recent dips / tree-finder artifacts);
  **93% still grow stars after the halo turnover, MORE than the control** (merger-
  driven, elevated *inner* growth). Dropping them is **not purely statistical** —
  distinct merger-active population (`c_200c` −0.36, higher `M*/Mh`, larger `R50`)
  — but profile *shape* is unbiased, so droppable **with a scope caveat**.
- **MAH-definition survey** (`doc/mah_definitions_research.md`): we already use

  tail** (fit to the running-max Mpeak, which captured the merger *spike*); real fix
  = a **summed-accreted-mass MAH from the merger trees**.
